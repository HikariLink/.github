# HikariLink / .github

本仓库是 **HikariLink** 组织的默认社区健康文件仓库，同时包含组织公共主页的 README 模版。

> This repository holds the default community health files and the public profile README for the **HikariLink** GitHub organization.

## 📁 仓库结构 · Structure

```
.github/
├── profile/
│   └── README.md        # 组织公共主页 · Organization public profile page
└── README.md            # 本说明文件 · This file
```

## 🔖 相关说明

- `profile/README.md` 会自动显示在 [HikariLink 组织主页](https://github.com/HikariLink) 上。
- 如需修改组织公共页面的内容，请直接编辑 `profile/README.md`。
- 欢迎通过 Pull Request 贡献改进！
